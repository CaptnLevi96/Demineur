Voici les règles

Le démineur est un jeu où vous devez découvrir toutes les cases d'une grille sans cliquer sur les mines. Chaque clic révèle le nombre de mines adjacentes à cette case. Utilisez ces informations pour éviter les mines et dégager toutes les cases sans mines. Si vous cliquez sur une mine, vous perdez la partie.
